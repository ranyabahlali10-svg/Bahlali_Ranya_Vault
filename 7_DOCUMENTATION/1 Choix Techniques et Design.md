# Choix Techniques et Esthétiques

Pour ce projet, j'ai cherché à créer une ambiance qui rappelle l'univers de *The Wire*.

### 1. Personnalisation CSS (CMS)
J'ai créé un snippet CSS (`the-wire-theme.css`) pour modifier l'interface :
- **Couleurs** : Utilisation du orange HBO pour les titres et les liens pour une identité visuelle forte.
- **Typographie** : Passage des titres en police pour simuler des rapports de police
### 2. Utilisation de l'HTML
J'ai injecté du code HTML directement dans les fiches des personnages principaux. Cela permet de créer des encadrés "Dossier Confidentiel" qui hiérarchisent l'information mieux que le simple Markdown. C'est ce qui donne l'aspect "CMS" (Content Management System) au Vault.
### 3. Littérate Programming
Comme demandé dans le TD, j'ai adopté une approche de **Literate Programming**. Mes notes ne sont pas juste du texte : elles intègrent du code (Dataview) qui génère du contenu dynamiquement. La note devient alors un mélange vivant de données techniques et d'analyse humaine.
