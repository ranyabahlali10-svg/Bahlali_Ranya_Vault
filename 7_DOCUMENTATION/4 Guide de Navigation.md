# Guide de Navigation

Pour explorer ce Vault de la manière la plus efficace, voici le parcours conseillé :

1. **La Page d'Accueil** : Commencer par le `README` à la racine pour comprendre le contexte.
2. **Le Canvas** : Ouvrir la vue graphique dans `5_VISUALISATIONS` pour voir la "toile d'araignée" de Baltimore.
3. **Les Fiches** : Naviguer de personnage en personnage via les liens internes `[[ ]]` pour voir comment les histoires s'entrecroisent.
4. **Les Slides** : Lancer la présentation intégrée pour avoir un résumé synthétique de la démarche.
