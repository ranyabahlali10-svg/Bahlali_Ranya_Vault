## Projet Obsidian - The Wire 
**Auteur :** Bahlali Ranya
**Cours :** Licence 3 - Web et Programmation

## Présentation du projet
Ce Vault Obsidian analyse la série *The Wire* comme un système complexe interconnectant la police, la rue, la politique et les institutions de Baltimore.

## Configuration Technique
Pour profiter pleinement de ce Vault, les plugins suivants sont nécessaires :
- **Dataview** (pour les tableaux dynamiques)
- **Advanced Slides** (pour la présentation intégrée)
- **Banners** (pour l'esthétique)

## Structure du Vault
- `0_META` : Templates et configuration.
- `1_PERSONNAGES` : Base de données des protagonistes.
- `7_DOCUMENTATION` : Explication de la méthodologie (Literate Programming).

## Comment lire ce projet ?
Commencez par ouvrir le fichier `PRESENTATION.md` et lancez le mode "Slide" pour une introduction guidée.
